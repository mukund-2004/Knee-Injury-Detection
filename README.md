# Knee-Injury-Detection

### •	Developed an AI-based knee MRI injury detection system using Deep Learning and PyTorch to classify ACL Tear, Meniscus Tear, and Abnormal cases from multi-view MRI scans (Axial, Coronal, and Sagittal).

### •	Implemented Grad-CAM visualization for explainable AI, enabling highlighted injury regions on MRI images through a Flask web application with real-time prediction and treatment recommendation support.
